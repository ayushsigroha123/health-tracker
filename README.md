# health-tracker


Comprehensive Health Tracker.

# Overview.

This Python program is pretty straightforward and easy for users to handle. It tracks different parts of personal health and analyzes them. The thing calculates BMI and puts it into categories. It also figures out daily water needs and calories burned each day. Plus it suggests steps to aim for and how much sleep to get. Based on what users enter it gives exercise ideas and food suggestions. People can log their health info from the day. They store it all in a CSV file and look back at old logs whenever they want.

# Features.

The program works out body mass index and sorts it into the right group. It estimates water intake for the day using your weight. Daily calorie requirements come from age gender height weight and how active you are. It picks a steps goal and sleep time based on age. Exercises get recommended that fit your BMI and age. Foods come suggested for whatever BMI category you fall into. Users log daily details right into a CSV file. They can pull up past logs from that same file. The interface runs on a simple menu so interaction stays easy.

# Technologies Tools Used.

It runs on Python 3.x. Standard libraries like datetime and csv handle the basics. No need for any outside dependencies at all.

# Installation Running.

Make sure Python 3 sits on your system. Save the code into a file like health_tracker.py. Open up a terminal or command prompt. Go to the folder with that file. Run it with python health_tracker.py.

# Instructions for Testing.

Pick option 1 from the menu to add your daily health info. Follow along with the prompts for name age gender weight height steps sleep hours and activity level. It shows the health metrics it calculated and the recommendations right there. All the data saves to comprehensive_health_log.csv. Go with option 2 to see every logged entry from before. Option 3 gets you out of the program. Test it out by putting in different data for various users. Check the outputs and what ends up in the logs.
